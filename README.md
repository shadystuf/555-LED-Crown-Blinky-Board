# 555-LED-Crown-Blinky-Board
I made this project based on one of the STASIS starter projects in a Hackclub workshop in St. Clare's, Oxford lead by Caterina. This was a great opportunity to understand how computers work, so I took it. This is my very first prject, so I wanted to start on something easier, like a blinky board.
